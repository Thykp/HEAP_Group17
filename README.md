Tech Stack:
Frontend: React, Tailwind css, Javascript, Typescript
Backend: Nodejs, Expressjs, Javascript
Database: Supabase
External API and Tools: OpenAI API, Shadcn/UI

To Run Locally:

Backend:
cd backend
npm i
npm run devstart

Backend should be running on localhost:3000

Frontend:
cd frontendV2
npm i
npm run dev

Frontend should be runnning on localhost:5173
